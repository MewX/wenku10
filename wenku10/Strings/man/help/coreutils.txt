coreutils

DESCRIPTION
	Common utilities
	Please note that these are not full-featured GNU utils. I just toke the names and implement a bare function.

Available Actions
	cat             Print file contents
	rm              Delete files or directories ( -r )
	mv              Move files or directories
	mkdir           Create directories
	wc              Print newline ( -l ), byte ( -c ) counts of a file
	ls              List directory contents
	cd              Change working directory
	pwd             Print the name of current working directory
	touch           Change file timestamps, creates sub-dirs if not exist

	uuidgen         Create a new uuid value
	date            Print the current date

	export          Export a file to somewhere
	import          Import a file from somewhere
