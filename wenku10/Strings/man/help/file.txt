File [ACTION] [TARGET]

DESCRIPTION
	Common file/directory operation commands.
	Please note that these are not fully featured GNU utils. I just take the names and implement a bare function.

Available Actions
	cat             Print file contents
	rm              Delete files or directories ( -r )
	wc              Print newline ( -l ), byte ( -c ) counts of a file
	ls              List directory contents
	cd              Change working directory
	pwd             Print the name of current working directory
	touch           Change file timestamps

	export          Export TARGET to somewhere
	import          Import data to TARGET
